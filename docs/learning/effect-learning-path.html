Effect 4 學習路徑

這些文章不是同一主題的重複版本，而是從語法、組合、錯誤、依賴到資源生命週期的不同層級。依序閱讀能避免一開始同時承擔太多概念。

版本基準：effect@4.0.0-beta.98；Bun 文章另使用 @effect/platform-bun@4.0.0-beta.98。

  2. pipe、map 與 flatMap
  3. 依需求選擇組合方式
  4. Effect 的建構與執行時間

第二階段：Typed errors

  3. cause 的公開型別邊界

第三階段：依賴與 Layer

  1. Context.Service 完整入門
  2. 以 Git service 練習 contract 與 Layer
  3. BunServices、provide 與 R = never

第四階段：資源、分支與 CLI

  1. Scope 與 Effect.scoped
  2. Operator 位置如何改變包裝邊界
  3. Match 與 exhaustive handling
  4. CLI Flags
  5. Argument、Flag 與 CliError

案例文章的定位

Git service、Bun build script 與 WeakMap error 都是把前面概念放進真實邊界的案例，不應拿來取代 Context、Scope 或 TaggedError 的基礎文章。
